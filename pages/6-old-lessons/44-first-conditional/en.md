# 44. If the weather is good (First Conditional)

## New words {new}

if
rain
snow
wind
cloud
umbrella
coat
hat
warm

## Affirmative (If + Present, Will + Verb)

If the weather is good, we will go to the park.
If it rains, I will stay at home.
If she comes, we will be happy.
If you work hard, you will have money.
If he buys a car, he will go to work.
If it is cold, I will wear a coat.
If the sun is hot, we will swim in the river.
If they help us, we will finish the work.
If I see him, I will give him the book.
If you wait, I will come with you.
If it snows, we will play outside.
If the wind is strong, we will stay home.
If she wears this hat, she will look beautiful.
If we take an umbrella, we will not be wet.
If it is warm, we will open the window.

## Negative (If + don't/doesn't, Won't + Verb)

If I don't sleep, I will be tired.
If you don't eat, you will be hungry.
If he doesn't work, he won't have money.
If she doesn't come, we won't go there.
If it doesn't rain, we will go to the forest.
If we don't buy food, we won't eat dinner.
If they don't study, they won't know English.
If the weather isn't good, we won't go outside.
If you don't take a coat, you will be cold.
If he doesn't help me, I won't finish it.
If I don't see her, I won't tell her.
If it doesn't snow, we won't play.
If the wind is strong, we won't go.
If she doesn't wear a hat, she will be cold.
If we don't have an umbrella, we will get wet.

## Questions (Will ... if ...?)

Will you go if I go?
Will he help us if we ask him?
Will she come if it rains?
Will they play if it is cold?
Will you be happy if you buy this?
Will we see him if we wait?
Will it be warm if the sun is hot?
Will you wear a coat if it is cold?
Will he take an umbrella if it rains?
Will she stay home if it snows?
Will they work if they are tired?
Will you tell me if you know?
Will we go to the park if the weather is good?
Will you call me if you are free?
Will he be angry if I don't come?
