## First mention - use a/an. Second mention - use the

ай сиː ə маунтəн.
з̣ə маунтəн из биг.
шиː хяз əн айлəнд.
з̣ə айлəнд из смоːл.
виː хяв ə ривə.
з̣ə ривə из лон̣.
з̣ей хяв ə форист.
з̣ə форист из биг.
хиː хяз əн амбрɛлə.
з̣ə амбрɛлə из нйуː.

## Unique things - use the

з̣ə сан из хот.
з̣ə скай из биг.
з̣ə сити из биг.
з̣ə скуːл из гуд.
з̣ə паːк из гуд.

## General things - use a/an

ə трейн из биг.
ə бридж из лон̣.
ə лейк из биг.
ə каː из гуд.
ə бук из гуд.
