# 48. If I were rich (Second Conditional)

## New words {new}

would
wish
dream
win, won, won
lottery
million
dollar
boss
rule
world

## Examples

If I were rich, I would buy a big house.
If I had more money, I would buy a car.
If we had more time, we would travel the world.
If she were free today, she would come.
If he had a new job, he would be happy.
If they were here, we would be happy.
If I had time, I would read more.
If you were my friend, you would help me.
If it were warm, we would go to the park.
If I were the boss, I would have one rule.
My dream is a big house.
I wish I had a dog.
If I win the lottery, I would buy a car.
If we had a million dollar house, we would be happy.
If she had a car, she would go to work.
