# 1. Я большой, ты маленький (to be)

## Личные местоимения и глагол to be. Смысл: "быть, являться, находиться".

я есть
ты, вы есть
он есть
она есть
оно есть
мы есть
они есть

## Сокращенные формы (означают то же самое, что и полные формы)

Я есть
ты, вы есть
он есть
она есть
оно есть
мы есть
они есть

маленький
высокий
низкий
хороший

горячий
холодный
старый
молодой
